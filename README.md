### There is a simple REST api

### Resource: User

### Endpoints:
- GET: /api/users
- POST: /api/users
- PUT: /api/users/{userId}
- PATCH: /api/users{userId}
- DELETE: /api/users{userId}
- GET: /api/users/range?fromDate=&toDate

### How to run
- run command 'docker compose up -d' to build database
- run app
